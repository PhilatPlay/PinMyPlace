üìã What is dropLogik?

dropLogik is a digital addressing service that creates permanent, scannable GPS addresses for locations without formal street addresses.

Core Problem Solved: Approximately 4 billion people (50% of global population) live in areas without formal street addresses. This prevents them from:

  • Receiving e-commerce deliveries
  • Getting mail and packages
  • Accessing emergency services
  • Participating in the digital economy

üéØ How It Works

  1. User pins exact GPS location on an interactive map (latitude/longitude coordinates)
  2. Payment via local methods (GCash, Maya, Stripe, Xendit) - approximately $3 USD
  3. Receive scannable QR code containing Google Maps directions to exact coordinates
  4. Share QR code with delivery services via WhatsApp, SMS, or print
  5. Delivery drivers scan the QR code and get GPS navigation directly to the location

üîß Technical Details

Technology Stack:

  • Frontend: Progressive Web App (HTML, CSS, JavaScript)
  • Backend: Node.js, Express
  • Database: MongoDB
  • Maps: Google Maps API
  • QR Generation: QRCode.js library
  • Payment Processing: Stripe (global), Xendit (Southeast Asia)
  • Hosting: AWS Elastic Beanstalk

Public API Available:

Base URL: https://droplogik.com/api

The API is publicly accessible without authentication (rate limits apply). Key endpoints:

  • POST /api/pin/initiate-payment - Create pin with payment
  • GET /api/pin/retrieve/:phone - Retrieve pins by phone number
  • POST /api/pin/create-with-code - Redeem bulk access codes
  • POST /api/bulk/purchase - Purchase bulk codes
  • GET /api/currency/detect - Auto-detect currency from IP

Rate Limits: 100 requests per 15 minutes per IP address

Documentation: droplogik.com/api.html

Data Stored per Pin:

  • GPS Coordinates (latitude/longitude with 6 decimal precision)
  • Location name/label
  • Optional address details (landmarks, floor numbers)
  • Phone number (for retrieval)
  • Google Maps URL (pre-generated for QR code)
  • Creation timestamp
  • Unique PIN ID
  • Optional drone delivery data (landing zone, obstacles, access restrictions, approach direction)
